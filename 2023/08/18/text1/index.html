hexo新建、编辑并预览文章

发表于2023-08-18|更新于2023-08-22
|阅读量:

新建

hexo new [layout] title

生成草稿文件夹和草稿

hexo n test test1

将文章推送到_posts下

hexo publish <test1>

生成静态文件

hexo g/generate

删除静态文件

hexo clean

启本地端口预览

hexo s/server

部署到博客

hexo d/deploy

插入图片格式

![alt](path 'title')
